# MusuiBot
ProxyBot Minecraft Bedrock that automatically pvp for you in the raw, literally from the target with the command /target <player>.

## Installation

Require [NodeJS](https://nodejs.org/fr/download)

In Terminal
```bash
  npm install
  node index or node .
```
